# 动漫角色素材目录

## 使用说明

将你的动漫角色图片放在这个目录下。

### 支持的格式

1. **静态图片**: `idle.png` (默认站立状态)
2. **多帧动画**: `idle_1.png`, `idle_2.png`, `idle_3.png` 等
3. **Live2D 模型**: 将 `.model3.json` 文件和相关资源放在子目录中

### 推荐尺寸

- 宽度: 150-250px
- 高度: 250-350px
- 格式: PNG (支持透明背景)

### 获取素材资源

- **免费资源**: 
  - [Pinterest](https://www.pinterest.com/) - 搜索 "anime character PNG"
  - [Pixiv](https://www.pixiv.net/) - 日本插画网站
  
- **Live2D 模型**:
  - [Live2D 官方示例](https://www.live2d.com/en/download/sample-data/)
  - [Booth](https://booth.pm/) - 日本数字内容市场

### 快速开始

如果还没有准备好素材,可以使用以下方式创建占位符:

1. 从网上下载一个喜欢的动漫角色 PNG 图片
2. 重命名为 `idle.png`
3. 放在当前目录
4. 重启应用即可看到效果
